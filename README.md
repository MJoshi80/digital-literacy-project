## Project Overview

This repository is my Digital Literacy Portfolio, submitted as part of the CSE0001 course at VIT Bhopal University. As a **Student Digital Ambassador**, I worked through five tasks that cover the key modules of the course — from creating digital awareness content, to building a professional online presence, to understanding cybercrime and online safety.

All tasks are documented in this repository. Each folder contains the relevant files, screenshots, and writeups for the corresponding module.

---

## Module Summaries

### Task 1 – Digital Literacy Awareness Infographic *(Module 1)*
Created a one-page infographic using **Canva** covering what digital literacy means, safe internet practices, and building a professional online presence. Exported and added to `task-1-presentation/`.

### Task 2 – Student Digital Portfolio *(Module 2)*
Set up profiles on **GitHub**, **LinkedIn**, and **Kaggle**. Created a GitHub Profile README and updated the LinkedIn Education section. Screenshots are in `task-2-portfolio/`.

### Task 3 – Coding & Collaboration Platforms *(Module 3)*
Completed a beginner challenge on **HackerRank** and created a **5-question Digital Literacy Awareness Quiz** using Google Forms.

🔗 **Google Form (Quiz) Link:** https://docs.google.com/forms/d/e/1FAIpQLSe8t8xcQogSPWk07vuyBccR2j846ErCLzx0pSHYbasmoEwyqw/viewform?usp=header

### Task 4 – Professional Email & Etiquette Guide *(Module 4)*
Wrote two professional emails (assignment extension request + internship inquiry) and created a social media checklist with 5 Do's and 5 Don'ts for college students.

### Task 5 – Cybercrime Awareness *(Module 5)*
Researched **Identity Theft** as a cybercrime. Wrote a 200–250 word case study and created a "Stay Safe Online" checklist with 8+ prevention tips, including UPI/financial safety and reporting channels.
